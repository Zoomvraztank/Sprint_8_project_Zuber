# Análisis de viajes y comportamiento de usuarios – Zuber

## Descripción del proyecto
Este proyecto analiza datos de viajes de la plataforma Zuber para identificar patrones de comportamiento de los usuarios y entender cómo factores externos (como el clima) afectan la duración de los viajes.

## Objetivo
Identificar diferencias en la duración de los viajes bajo distintas condiciones y validar si estas diferencias son estadísticamente significativas.

## Tecnologías utilizadas
- Python (pandas, numpy)
- SQL
- Matplotlib / visualización de datos
- Estadística (prueba t)

## Análisis realizado
- Limpieza y preparación de datos
- Análisis exploratorio de datos (EDA)
- Consultas SQL para agregación y segmentación
- Comparación de grupos de datos
- Aplicación de prueba t para validación de hipótesis

## Resultados
- Se identificaron diferencias en la duración de los viajes dependiendo de las condiciones climáticas
- Se comprobó estadísticamente que estas diferencias son significativas
- Se detectaron patrones relevantes en el comportamiento de los usuarios

## Conclusiones
Las condiciones externas influyen directamente en la duración de los viajes, lo que puede impactar la experiencia del usuario y la operación del servicio.

## Aprendizajes
- Uso de SQL para análisis de datos reales
- Aplicación de pruebas estadísticas en escenarios reales
- Interpretación de datos para toma de decisiones
