# 🚕 Sprint 8 - Zuber Data Analysis

## 📌 Descripción del proyecto
En este proyecto analizo datos de una empresa de transporte tipo Uber (Zuber) en Chicago.

El objetivo es:
- entender el comportamiento de los viajes
- analizar las empresas de taxis
- identificar patrones en los barrios
- evaluar el impacto del clima en la duración de los viajes

---

## 📊 Datos utilizados

Se trabajó con 3 datasets:

- `project_sql_result_01.csv` → número de viajes por empresa
- `project_sql_result_04.csv` → promedio de viajes por barrio
- `project_sql_result_07.csv` → viajes con clima y duración

---

## ⚙️ Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- SciPy
- Jupyter Notebook

---

## 🔍 Análisis realizado

- limpieza y revisión de datos
- análisis exploratorio (EDA)
- identificación de top 10 barrios
- visualización de datos
- prueba de hipótesis estadística

---
